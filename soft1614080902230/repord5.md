# 1.实验目标

掌握在Android Studio中存储数据

# 2.实验内容（选择文件内部存储）

（1）将应用产生的数据保存到文件存储中

（2）将运行即如果截图

# 3.实验步骤

（1）启动Android Studio打开项目soft1614080902230

（2）将上一次的secondActivity里的代码修改，做成文件存储和读取功能的界面

（3）添加按钮控件和文本控件，并且给两个按钮注册监听器，以触发保存和读取数据的监听事件

（4）写一段数据，点击保存按钮，屏幕下方显示“数据保存成功”，然后点击读取按钮，下方显示刚才保存的数据。这里指的注意的事，由于代码中没有删除的操作，因此每次保存数据会追加到上一次保存数据后面，不会覆盖，因此再一次点击读取按钮的时候看到的是第一二次保存的数据

# 5.实验体会

   这次实验的重点在于对java语法的理解，Android Studio里的文件存储是Android中最基本的一种数据存储方式，它与Java中的文件存储类似，都是通过I/O流的形式把数据原封不动的存储到文档中。代码中要先定义初始化变量，要让控件id和其对应的初始变量对应准确，否则会导致后面连锁性错误。这里在写监控事件时应用了switch..case..语句，是对两个按钮分别进行书写监控事件的代码，清晰明了。做实验还要注意布局，因为一开始没有加布局，只加了控件，导致导出apk的时候界面控件的位置和编写时不一样，会错乱，这样会影响apk的使用。
