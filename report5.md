# 第五次实验

## 实验目的
掌握在Android App中存储数据

## 实验要求
+ 1.将应用产生的数据保存到文件存储中；
+ 2.使用内部文件存储方式。
+ 3.将运行结果截图。

## 实验步骤
+ 1.添加文件保存、更新等BUTTON。
+ 2.编写代码，使用数据流将保存里的影视剧链接数据输出到内部文件"file_leaderboard.txt"
+ 3.并且使用保存内部文件"file_leaderboard.txt"将百度云盘里的资源链接数据更新
+ 4.使用Git将文件提交到库：https://github.com/dencyu/android-labs-2018；
+ 5.在自己的GitHub库上创建和发送Pull Request；
+ 6.编写实验报告。
