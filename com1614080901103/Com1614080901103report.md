    （2）.加入Git项目（Fork），将项目克隆（Clone）到自己帐号下；
          $ git clone https://github.com/CAIYB/android-labs-2018/
    （3）.创建以学号命名的个人实验目录：
          com161408901103。
    （4）.创建一个Java类：edu.hzuapps.androidlabs.com1614080901103.Com1614080901103Activity.java。
    （5）.提交代码并发Pull Request，老师合并之后截图。

## 4. 实验体会

**这次的实验主要是先熟悉github这个平台的使用，以及对git的使用，在其中操作时有时会出现各种乱七八糟的错误，然后就是不断地修改
提交的文件直到老师合并通过。其实所做的作业并不多也不难，但是就是对于这些的使用不是很熟悉，对于一些规范不是很懂，这就造成不
断的需要修改。做实验的时候会和同学一起去讨论，这样就减少了修改次数以及增加更多的知识。**
