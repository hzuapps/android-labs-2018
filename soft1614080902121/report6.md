# 实验目的

1.掌握Android网络访问方法；

2.理解XML和JSON表示数据的方法。 
 
# 实验要求

1.从网络下载一个文件（图片、MP3、MP4）；

2.保存到手机，在应用中使用文件；

3.将应用运行结果截图。

# 实验步骤

1.先设计好从网络下载文件的页面的布局。

2.通过使用线程、URL和文件输入输出流进行下载网络图片的类的编程。

3.在AndroidManifest.xml文件中添加写入SD卡权限和网络连接权限。
