# 实验六

## 一、实验目的
  掌握Android网络访问方法<br>
  
## 二、实验内容
  1、从网络下载一个文件（图片、MP3、MP4）；
  2、保存到手机，在应用中使用文件；
  3、将应用运行结果截图。

## 三、实验步骤
  1、创建dowmloadActivity<br>
  2、设置下载按钮<br>
  3、加载网络地址<br>
  4、保存图片到本地<br>
  5、提交代码并发Pull Request： $ git pull $ git add 路径/项目/* $ git commit "#6 #970 第六次实验" $ git push<br>
