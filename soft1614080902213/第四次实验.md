# 第四次实验

## 1. 实验目标

- （1）根据选题要求设计界面布局及控件使用；
- （2）布局 xml 及界面控制操作代码提交并截图；
- （3）将应用运行结果截图。

## 2. 实验步骤

- （1）在首页添加了添加日记按钮，点击后跳转到日记页面。
- （2）在日记页面 ConstraintLayout 里分别创建了一个 LinearLayout 和一个 RelativeLayout。
- （3）在日记页面增加了保存和取消这 两个按钮，点击后分别执行保存日记到txt文件和退出操作。
- （4）在日记页面增加了一个 EditText，用于储存文本内容。
