# 第六次实验

## 1. 实验要求
1. 从网络下载一个文件（图片、MP3、MP4）；
2. 保存到手机，在应用中使用文件；
3. 将应用运行结果截图。

## 2. 实验步骤
1. 先修改AndroidManifest.xml中的文件，以获取上网的相关权限；
2. 在Donwload.java文件中实现连接网络获取图片，显示图片，图片保存功能；
3. 定义connectNet类用以下载图片，new 一个线程Thread进行下载；
4. 使用加载保存按钮获取URL的图片并保存；
5. 用git上传到github。

## 3. 实验结果
输入一个图片的url地址然后点击保存加载，成功加载到一个图片。
